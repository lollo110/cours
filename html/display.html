Display: block

Cette div est un élément en Block. Il occupe toute la largeur de l'écran.

Exemple de balises de type 'block' :

  • div
  • p
  • h1,h2,h3,h4,h5,h6
  • li
  • main
  • ul,ol

Display: Inline

Les éléments en inline ne forcent pas le retour à la ligne, il prend simplement la place dont il a besoin.

Attention !! les éléments de type inline ne sont pas comme les type block, leur hauteur et largeur ne sont délimités QUE par leur contenu.

Exemple de balises de type inline

  • a (liens)
  • img
  • input
  • video
  • label
  • span

Display : inline-block

Inline-block: C'est un type inline aui conserve les propriétés du type block, il peur donc être modifié en hauteur et largeur comme n'importe quel type block.

Display: Flex

Flex : les éléments dans le conteneur s'alignent en ligne et se répartissent l'espace disponible.

Display : Grid

Grid : Avec grid, on peut définir une grille avec des lignes et des colonnes

Display : none

None : Le display none cache tout simplement l'élément.
